Usage: gitcm.sh {options}

This is git config manage script.

你可以使用如下参数进行操作
-c                                           查看当前配置
-d {config_name}                             删除指定配置
-e {config_name} {key} {value}               编辑指定配置
-g {config_name}                             获取指定配置
-h                                           输出帮助信息
-l                                           输出配置列表
-i {config_name} {user_name} {user_email}    初始化新配置
-u {config_name}                             切换配置
-v                                           输出调试信息
-?                                           输出帮助信息
